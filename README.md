
# DEPI – Round 4

## Data Analytics Datasets & Python Notebooks Repository

Welcome to the **DEPI Round 4 Data Analytics Repository**.
This repository contains **datasets and Python notebooks** used in the **Digital Egypt Pioneers Initiative (DEPI)** training program for Round 4 participants.

The goal of this repository is to provide students with **hands-on practice materials** for learning **data analysis using Python**.

---

## 📁 Repository Contents

This repository includes two main components:

### 1. Datasets

A collection of datasets used in exercises, demonstrations, and assignments during the course.

Examples include:

* Economic datasets
* Business and sales data
* Population and demographic data
* Public datasets used in analytics training

Datasets are typically stored in:

```
datasets/
```

Common formats include:

* CSV
* Excel
* JSON

---

### 2. Python Notebooks

The repository also contains **Jupyter notebooks** demonstrating various data analysis techniques using Python.

Notebooks are located in:

```
notebooks/
```

Topics covered in the notebooks include:

* Python fundamentals for data analysis
* Working with tabular data
* Data manipulation using Pandas
* Numerical computing with NumPy
* Data visualization with Matplotlib
* Exploratory Data Analysis (EDA)

---

## 🧰 Technologies Used

The notebooks in this repository primarily use the following Python libraries:

* Python 3
* NumPy
* Pandas
* Matplotlib

These tools form the **core toolkit for modern data analytics**.

---

## 🚀 How to Use This Repository

### Option 1 — Run in Google Colab

1. Open the notebook in this repository.
2. Click **Open in Colab**.
3. Run the notebook cells step-by-step.

This allows you to execute the notebooks **without installing anything locally**.

---

### Option 2 — Run Locally

1. Clone the repository

```
git clone https://github.com/<repo-name>.git
```

2. Navigate to the project folder

```
cd <repo-name>
```

3. Launch Jupyter Notebook

```
jupyter notebook
```
